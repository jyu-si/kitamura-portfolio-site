# Project Rules

## Purpose

- このサイトは、北村周仁の個人ポートフォリオサイトである。
- 就職活動や面談で共有するためのWebポートフォリオとして制作する。
- 特定企業向けの内容にはせず、制作物、研究、学生自治委員会での活動を見せる。

## Message and Concept

- 中心メッセージは「ものづくりを通して『ユーザーの体験を豊かにできる』エンジニア」とする。
- 補助コンセプトは「人の困りごとを、技術と仕組みで形にする」とする。

## Content Structure

- Visionセクションを削除しない。
- Visionの `Can / Will / Must` 構成を維持する。
- Projectsは「和大DE話題」「わらえんじぇる」「和歌山大学での研究」「学生自治委員会での活動」の4つを基本構成とする。
- VRラボツアーは独立Projectにせず、「学生自治委員会での活動」の中に含める。
- ESのような長文の自己PRにはせず、見出しと短文を使って分かりやすく伝える。

## Design

- 白背景、青系アクセント、十分な余白、カード型レイアウトを基本にする。

## Data and Assets

- Project情報は `src/data/projects.ts` にまとめる。
- Profile情報は `src/data/profile.ts` にまとめる。
- 画像は `public/assets/` 以下を参照する。
- 必要な画像が不足している場合は、代替画像を勝手に捏造せず、不足しているパスを報告する。

## Verification

- 作業後は `npm run build` を実行する。
